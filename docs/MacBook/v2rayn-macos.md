### 第二步：安装与报错修复
1. 打开下载好的 `.dmg` 安装包，将 **「v2rayN 图标」** 拖动到右侧的 **「Applications (应用程序)」** 文件夹中即可完成安装。
2. 前往 Mac 的启动台打开 v2rayN。

!!! danger "⚠️ 高频报错解决：提示『已损坏，无法打开』"
    由于苹果系统的安全保护机制，首次打开时极可能会提示软件“已损坏”。**千万不要删除！请按以下步骤一键修复：**
    
    1. 在报错弹窗上点击 **「取消」**。
    2. 按下键盘 `Command + 空格`，搜索并打开 **「终端 (Terminal)」**。
    3. 复制以下命令粘贴到终端中，然后按回车：
       ```bash
       xattr -cr /Applications/v2rayN.app
       ```
    4. 终端会提示输入密码，**请直接盲打输入您的 Mac 开机密码并回车**（输入时屏幕不会显示字符是正常现象）。
    5. 修复完成后，重新打开软件即可畅通无阻！

## 🚦 4. 开启代理与路由模式

有了节点并拉取出来后，还需要开启代理开关才能正式上网：

### 第一步：选择心仪节点
在主界面下方会出现一排排服务器节点。右键点击全部节点，选择 **“测试服务器真连接延迟”**。
选中一个延迟数字较小（网速较快）的节点，选择某一条线路**鼠标右键 -> 选择「设为活动服务器」**（此时该节点会变色或带有“活动”标记）。

### 第二步：开启系统代理
1. 在 Mac 屏幕顶部状态栏（或右下角）找到 **v2rayN 的小图标**。
2. **点击右键**，在菜单中勾选 **「自动配置系统代理」**，路由模式选择**「绕过大陆或全局」**
3. **✅ 连接成功标志：** 观察图标颜色，当软件图标**从蓝色变成红色**时，代表代理已成功开启！您可以打开 [Google.com](https://google.com) 进行测试。

### 第三步：认识路由分流模式
为了让上网更智能，请在软件的“路由”设置中选择合适的模式：
* 🎯 **绕过大陆 (Whitelist) [强烈推荐]：** 国内网站直连，国外网站走代理。既省流量又不影响国内应用（如微信、淘宝）的网速。
* 🌐 **全局 (Global)：** 所有流量全部强制走海外节点。仅在遇到特殊网页打不开时临时使用。

??? tip "🔥 进阶极客玩法：如何开启 TUN 模式？（全流量接管）"
    如果您发现某些电报软件 (Telegram) 或游戏无法翻墙，请开启更强大的 **Tun 模式**（创建一个虚拟网卡，强制接管电脑所有流量）：
    
    1. **设置授权密码：** 点击顶部菜单 **「设置」 -> 「参数设置」 -> 「Tun 模式设置」** 标签页。在密码框输入你的 **Mac 电脑开机密码** 并保存（此密码仅保存在本地用于授权网卡）。
    2. **启动 Tun 模式：** 在主界面或顶部菜单栏勾选 **「开启 Tun 模式」**。
    3. ⚠️ **最重要的一步：** 开启 Tun 后，请务必在菜单栏右键点击 **「清除系统代理」**！让虚拟网卡全面接管网络，避免代理冲突。
    4. *(注：Tun 模式下，“绕过大陆”路由规则依然有效，可完美实现智能分流。平日不需要接管全局时，建议关掉 Tun 换回“自动配置系统代理”以获得更快的速度。)*

## 🔄 5. 更新订阅与常见故障排查

中国大陆网络环境复杂，**翻儿Cloud 会定期维护和更换节点 IP**。当您发现节点无法上网或大面积超时时，请务必执行更新操作：

### 如何更新拉取最新节点？
1. 点击 v2rayN 顶部菜单栏的 **「订阅分组」**。
2. 点击 **「更新全部订阅 (不通过代理)」**。
3. 稍等几秒钟，主界面的节点列表就会自动刷新为翻儿Cloud的最新可用线路。

### 🛠️ 故障排查：更新失败怎么办？
如果您在更新时毫无反应或提示失败，请按照以下步骤排查：

1. **走代理更新：** 可能是因为订阅域名被当地运营商阻断。请先随意连接一个**旧的可用节点**，开启代理后，在菜单中选择 **「更新全部订阅 (通过代理)」** 进行强制更新。
2. **检查时间与链接：** 确保您的电脑系统时间精准无误；若仍不行，请前往官网重新复制一遍最新的订阅链接并修改替换。
3. **重启重试：** 在软件主界面尝试点击 **“重启服务”** 解决内核卡死问题，或尝试连接手机热点后再进行更新。
