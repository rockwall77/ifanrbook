# 🔄 v2rayN for macOS 订阅更新与故障排查 (2026 版)

!!! warning "开始前必读（防错检查）"
    为了确保节点更新顺利，请确认以下环境：
    
    - ✅ 浏览器访问 [time.is](https://time.is) 检查 **Mac 电脑本地时间** 是否与北京时间完全同步。（时间误差超过1分钟会导致更新彻底失败）
    - ✅ 请确保您的 **翻儿Cloud** 会员套餐未过期且流量充足。
    - ✅ 请确保当前网络未被公司/校园严苛内网限制。
    - ✅ 请确保您的 **翻儿Cloud** 会员套餐处于有效期内。

## 📡 1. 常规更新订阅教程

中国大陆的网络防火墙（GFW）时刻在变动，**翻儿Cloud 会定期维护与扩容服务器节点**。当您发现大面积节点显示超时、延迟过高，或完全无法打开网页时，请务必及时更新您的订阅配置以拉取最新线路：

1. 打开 Mac 上的 v2rayN 客户端主界面。
2. 将鼠标移动到顶部菜单栏，点击 **「订阅分组」**。
3. 在下拉菜单中选择 **「更新全部订阅 (不通过代理)」**。
4. 稍等 3~5 秒钟，主界面下方的服务器列表就会自动刷新，呈现出翻儿Cloud最新的国家与地区线路！

!!! success "更新后的操作：测速与重连"
    更新完成后，建议**全选（Command + A）**节点列表，右键选择 **「测试服务器真连接延迟」**。随后选中一个延迟最低的绿色节点，右键点击 **「设为活动服务器」** 即可畅享极速网络。

## 🛠️ 2. 故障排除：更新失败怎么办？

由于国内网络环境复杂，有时会出现点击“更新”后毫无反应，或直接弹窗提示“网络超时”、“未能解析域名”的情况。这通常是因为**翻儿Cloud的订阅域名被您当地的网络运营商临时阻断了**。

!!! tip "终极解决办法：走代理强制更新"
    如果遇到无法更新的情况，请执行以下“先上车，再买票”的神级操作：
    
    1. **先连旧节点：** 在软件主界面，随便找一个**当前还能勉强连得上**的旧节点，设为活动服务器。
    2. **确认翻墙状态：** 确保右下角系统托盘的 v2rayN 图标是**红色（已开启自动配置系统代理）**，且能打开谷歌。
    3. **切换更新模式：** 在翻墙状态下，再次点击顶部菜单栏的 **「订阅分组」**。
    4. **走代理更新：** 这一次请务必选择 **「更新全部订阅 (通过代理)」**！
    
    *此时更新请求将通过海外节点加密传输，100% 可以绕过国内运营商的网络阻断，成功拉取最新配置！*

### 备用排障方案
如果连旧节点都全部失效了，无法走代理更新，请尝试：
* **切换网络环境：** 暂时断开 Mac 的 WiFi，使用手机开启 **4G/5G 个人热点** 连接电脑后，再次尝试常规更新。
* **重置订阅链接：** 登录 [翻儿Cloud 官网](https://zifanr.net/)，重新复制一遍最新的 v2ray 订阅链接，在软件中点击“➕”号修改并替换掉旧的链接。
