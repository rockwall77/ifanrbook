# 🔄 macOS Clash Verge 订阅更新与维护 (2026 版)

!!! warning "开始前必读（防错检查）"
    为了确保节点更新顺利，请确认以下环境：
    

    - ✅ 浏览器访问 [time.is](https://time.is) 检查**Mac 电脑时间**是否与北京时间完全同步。
    - ✅ 请确保当前网络未被公司/校园严苛内网限制。
    - ✅ 若更新失败，建议开启 TUN 模式或切换为“全局”模式再试。

---

## 📡 1. 更新订阅配置

翻儿Cloud 会定期维护与扩容服务器节点。若发现节点大面积超时或无法连接，请务必及时更新您的专属订阅：

1. 打开 Mac 上的 Clash Verge 客户端，进入左侧的 **「订阅 (Profiles)」** 页面。
2. 找到您导入的 `翻儿Cloud` 配置文件卡片。
3. 点击卡片右上角的 **「半圆形刷新箭头」** 图标。
4. 等待上方进度条跑完，节点列表即会自动同步至最新状态。

<div align="center">
  <img src="https://ifanr.org/file/e6e8865181edb849cceae.png" alt="更新订阅" width="80%">
</div>


!!! success "自动化维护小贴士"
    强烈建议在导入订阅时设置好 **“更新间隔”**（如 `1440` 分钟），这样客户端会在后台默默为您自动拉取最新线路，省去手动更新的烦恼！

## 🛠️ 2. 在线升级客户端版本

保持客户端处于最新版本，能够获得更好的网络协议支持与更低的内核占用。Clash Verge 支持应用内直接平滑升级：

1. 注意观察桌面右上角或程序坞中客户端图标的左上角，若出现红色的 **“NEW”** 字样，表示有新版本可供升级。
2. 点击带有“NEW”提示的图标，系统会弹出升级提示页面。
3. 直接在此页面点击 **「更新 (Update)」**。
4. 下载完成后，软件会自动重启并应用最新版本。
