# 🔄 ClashX Pro 订阅更新与维护 (2026 版)

!!! warning "开始前必读（防错检查）"
    为了确保节点更新顺利，请确认以下环境：
    
    - ✅ 浏览器访问 [time.is](https://time.is) 检查 **Mac 电脑时间** 是否与北京时间完全同步。
    - ✅ 请确保当前网络未被公司/校园严苛内网限制。
    - ✅ 请确保您的 **翻儿Cloud** 会员套餐处于有效期内。

---

## 📡 1. 常规更新订阅教程

翻儿Cloud 会定期对服务器节点进行维护与扩容。如果发现大面积节点不可用或显示超时，请务必及时更新您的订阅配置以获取最新线路：

1. 点击 Mac 屏幕顶部状态栏的 **ClashX Pro 小猫咪图标**，展开主菜单。
2. 依次选择 **「配置」 -> 「托管配置」 -> 「管理」**。
3. 在弹出的窗口中找到您的 `翻儿Cloud` 配置文件，点击其后方的 **「更新」** 按钮。
4. 等待几秒钟，提示更新成功后，回到主菜单查看节点列表，即会同步为最新状态！

    建议在「托管配置管理」窗口中，勾选 **“自动更新”**，并将 **“更新间隔”** 设置为 `24` 小时。这样客户端就会在后台默默为您拉取最新线路，一劳永逸！

## 🛠️ 2. 故障排除：更新失败如何操作？

由于国内网络防火墙（GFW）的干扰，有时会出现点击更新后毫无反应，或提示“网络超时”的情况。

!!! tip "终极解决办法：走代理强制更新"
    如果出现更新失败，请尝试“先上车，再买票”的操作：
    
    1. 在状态栏菜单中，先随便选择一个**当前还能连得上**的旧节点。
    2. 确保勾选了 **「设置为系统代理」** 和 **「增强模式」**，使电脑处于“翻墙”状态。
    3. 在翻墙状态下，再次前往“托管配置 -> 管理”执行更新操作。
    4. 此时更新请求将通过海外节点加密传输，100% 可以绕过大陆网络阻断，成功拉取最新配置！
