# 🔄 v2rayNG 订阅更新与故障排查 (2026 版)

!!! warning "开始前必读（防错检查）"
    为了确保节点更新顺利，请确认以下环境：
    
    - ✅ 浏览器访问 [time.is](https://time.is) 检查**手机本地时间**是否与北京时间完全同步。
    - ✅ 请确保当前网络未被公司/校园严苛内网限制。
    - ✅ 请确保您的 **翻儿Cloud** 会员套餐处于有效期内。

## 📡 1. 常规更新订阅教程

翻儿Cloud 会定期维护与扩容服务器节点。若发现节点大面积不可用或显示超时，请务必及时更新您的订阅地址以获取最新线路：

1. 打开手机上的 v2rayNG 客户端，进入主界面。
2. 点击右上角的 **「三个点 (⋮)」** 更多按钮。
3. 在弹出的菜单选项中点击 **「更新订阅」**。
4. 稍等片刻，屏幕下方会提示更新成功，并刷出翻儿Cloud最新的国家地区节点列表。

## 🛠️ 2. 故障排除：更新失败如何操作？

由于国内网络环境复杂，有时会出现点击更新后毫无反应，或提示“网络超时”、“无法连接”的情况。

!!! tip "遇到更新失败请尝试以下方法"
    1. **切换网络环境：** 暂时断开家里的 WiFi，使用手机的 **4G/5G 数据流量** 再次尝试点击更新。
    2. **走代理更新：** 
       * 先随便选择一个**当前还能连得上**的旧节点。
       * 点击右下角按钮**启动 VPN 连接**。
       * 在翻墙状态下，再次点击右上角执行「更新订阅」。通过海外节点加密拉取，通常能 100% 成功！
