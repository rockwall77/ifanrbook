# 🔄 Surfboard 订阅更新与故障排查 (2026 版)

!!! warning "开始前必读（防错检查）"
    为了确保节点更新顺利，请确认以下环境：
    
    - ✅ 浏览器访问 [time.is](https://time.is) 检查**手机本地时间**是否与北京时间完全同步。
    - ✅ 请确保当前网络未被公司/校园严苛内网限制。
    - ✅ 请确保您的 **翻儿Cloud** 会员套餐处于有效期内。

## 📡 1. 常规更新订阅教程

翻儿Cloud 会定期维护与扩容服务器节点。若发现节点大面积不可用或显示超时，请务必及时更新您的配置文件以获取最新线路：

1. 打开 Surfboard 客户端，底部菜单点击进入 **「配置 (Profiles)」** 页面。
2. 找到您正在使用的 `翻儿Cloud` 配置文件。
3. 点击该配置文件最右侧的 **「竖排三个点 (⋮)」** 更多按钮。
4. 在弹出的菜单中选择 **「检查更新 (Check update)」**。
5. 屏幕下方会提示正在更新，更新完成后，节点列表将自动同步为翻儿Cloud的最新状态！

## 🛠️ 2. 故障排除：更新失败如何操作？

由于国内网络防火墙（GFW）的干扰，有时会出现点击更新后毫无反应，或提示“网络超时”、“更新失败”的情况。

!!! tip "遇到更新失败请尝试以下方法"
    1. **切换网络环境：** 暂时断开家里的 WiFi，使用手机的 **4G/5G 数据流量** 再次尝试点击更新。
    2. **走代理更新：** 
       * 先进入「代理」页面，随便选择一个**当前还能连得上**的旧节点。
       * 回到「仪表盘」点击右下角按钮**启动 VPN 连接**。
       * 在翻墙状态下，再次回到「配置」页面执行「检查更新」。通过海外节点加密拉取，通常能 100% 成功绕过阻断！
