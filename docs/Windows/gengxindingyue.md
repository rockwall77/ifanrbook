# 🔄 Clash Verge 订阅更新与故障排除 (2026 版)

!!! warning "开始前必读（防错检查）"
    为了确保节点更新顺利，请确认以下环境：
    
    - ✅ 浏览器访问 [time.is](https://time.is) 检查**设备本地时间**是否与北京时间完全同步。
    - ✅ 请确保网络未被公司/校园内网限制。
    - ✅ 若更新失败，建议开启 TUN 模式或切换“绕过大陆”模式重试。

---

## 📡 1. 更新订阅配置

翻儿Cloud 会定期维护与扩容节点。若发现节点无法连接或显示超时，请及时更新您的专属订阅：

1. 打开 Clash Verge 客户端，进入左侧的 **「订阅」** 页面。
2. 选中您的专属配置文件，**点击鼠标右键**。
3. 选择 **「更新」** 或 **「更新 (通过代理)」**，等待进度条完成，节点列表将自动刷新。

## 🛠️ 2. 常见问题排查

若更新订阅失败，请尝试以下步骤：

!!! tip "故障排除建议"
    1. **检查代理状态：** 确保已开启「系统代理」模式后再尝试更新订阅。
    2. **重置订阅：** 若问题依旧，请在「订阅」页面右键删除旧的 `.yaml` 配置文件，并按照基础配置教程重新导入专属订阅链接。
