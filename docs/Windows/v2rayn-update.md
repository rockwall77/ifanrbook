# 🔄 v2rayN 订阅更新与内核维护 (2026 版)

!!! warning "开始前必读（防错检查）"
    为了确保节点更新顺利，请确认以下环境：

    - ✅ 浏览器访问 [time.is](https://time.is) 检查设备时间是否与北京时间同步。
    - ✅ 请确保网络未被公司/校园内网限制。
    - ✅ 若订阅更新失败，建议开启 TUN 模式或切换“绕过大陆”模式重试。

---

## 📡 1. 订阅更新

翻儿Cloud 会定期维护与扩容节点。若发现节点无法连接或出现超时，请务必执行以下操作获取最新线路：

1. 打开 v2rayN 主界面。
2. 点击顶部菜单 **「订阅分组」** -> **「更新全部订阅 (不通过代理)」**。
3. 若此时提示失败，请稍后重试，并选择 **「更新全部订阅 (通过代理)」**。
4. 等待界面刷新，出现最新的节点列表后，即可正常使用。

## 🛠️ 2. 软件与核心维护

除了更新订阅，定期更新软件内核也是保障连接稳定性的关键：

1. 在 v2rayN 主界面点击 **「检查更新」**。
2. 分别点击以下选项进行升级：
    - **检查更新 v2rayN** (主程序)
    - **检查更新 Xray-Core / sing-box Core** (底层核心引擎)
    - **检查更新 Geo files** (核心路由规则文件)
3. 下载完成后，程序会自动完成替换与重启。

!!! success "自动化维护小贴士"
    建议进入 v2rayN 的 **「设置」**，勾选 **“启动时自动更新订阅”**。这样您每次开机启动软件时，无需手动操作即可自动同步官方最新的服务器线路！
