# 🔄 Stash 订阅更新与故障排查 (2026 版)

!!! warning "开始前必读（防错检查）"
    为了确保节点更新顺利，请确认以下环境：
    
    - ✅ 浏览器访问 [time.is](https://time.is) 检查**手机本地时间**是否与北京时间完全同步。
    - ✅ 请确保当前网络未被公司/校园严苛内网限制。
    - ✅ 请确保您的 **翻儿Cloud** 会员套餐处于有效期内。

## 📡 1. 常规更新订阅教程

翻儿Cloud 会定期维护与扩容服务器节点。当科学上网出现障碍或有非常多的线路显示超时，请务必及时更新您的配置文件以获取最新节点信息。

=== "设置自动更新 (强烈推荐)"

    1. 在 Stash 客户端底栏，点击进入 **「设置」** 页面。
    2. 滑动找到 **「更多设置」**。
    3. 将 **「更新间隔」** 设置为 `24` 或 `12`（小时单位）。
    4. 确保开启 **「自动更新」** 开关。
    5. 设置完毕后，只要打开 App，系统会在后台为您自动拉取最新线路，一劳永逸！

=== "手动强制更新"

    1. 在 Stash 首页底栏，进入 **「设置」 -> 「配置文件」**。
    2. 找到您正在使用的 `翻儿Cloud` 配置文件。
    3. **向左滑动**该配置文件的名称。
    4. 在弹出的菜单中点击 **「更新」**。

## 🛠️ 2. 故障排除：更新失败如何操作？

由于国内网络防火墙（GFW）的干扰，有时会出现点击更新后毫无反应，或提示“网络超时”、“解析失败”的情况。这通常是因为**订阅域名被大陆网络临时阻断了**。

!!! tip "终极解决办法：走代理强制更新"
    如果出现更新失败，请执行以下操作：
    
    1. 在 Stash 首页，随便选择一个**当前还能连得上**的旧节点。
    2. 点击首页顶部的 **「启动」** 按钮，确保手机已经处于“翻墙”状态（状态栏出现 VPN 图标）。
    3. 在翻墙状态下，再次前往配置文件列表执行“向左滑 -> 更新”操作。
    4. 此时更新请求将通过海外节点加密传输，100% 可以绕过网络阻断，成功拉取最新配置！
